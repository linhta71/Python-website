So how does for loop work?

For loops are traditionally used when you have a block of code which you want to repeat a fixed number of times. The Python for statement iterates over the members of a sequence in order, executing the block each time.

The general command of a for loop:

for var in range(n):

...chunk of commands...

Explanation for for loop:

  1. for var in range(n): this line of code is used to command that everything that is indented and lie under it will be replicated for n times.
    1. var:This is an iterator. This can be replaced by any letters or words.
        2. For example: var can be replaced by a letter: a or b or c, etc, or by a word such as time, or character , etc.
    2. n: number of times that the chunk of code should be replicated. This must be an integer.

    Notice: varvar only takes value from 0 to n-1.

For example: changing the general “for loop” above into a real “for loop” that draws a square will be:

  1. If you choose your var is i, then you need to replace var to i.
  2. There are 4 equal edges in a square.
  3. There are 4 angles in a square, each angle is 90 degrees. If you decide to draw the square from the left, then you will turn your turtle to the left for 90 degrees.
  4. From (b) and (c) you will know that you have to repeat the code that draws the edge and the angle of the square for 4 times in order to create a square.
  5. Then here is the “for loop” for drawing the square.

    6. for i in range(4):

    name.forward(length) (draw edge)

    name.left(90) (draw angle)

Notice:Indentation is very important in a for loop of a Python program. It helps indicate which chunk or line of code must be replicated and which line or chunk of code should not be. Without precise indentation, code script will not work.

Part 2: Now let’s draw a simple house by using for loop.

Notice:
  1. A house must have a body and a roof on top of the body. You don’t need to draw any additional elements in the house.
  2. Your turtle’s color will automatically be the house color.
  3. The body of the house is a square. Use a for loop to draw the body.
  4. The roof of the house will be an equilateral triangle. Use a for loop to draw the roof.
  5. All the edges of the house are the same, which means, all edges of the square body and triangle rooftop are the same.
  6. There are 2 separate for loops in this program, one to draw the square and one to draw the triangle.
